Escopo de Serviço
Landing page que
converte — e é sua.

Desenvolvemos a página do zero: design com a identidade do cliente, copy alinhada à oferta, rastreamento correto e hospedagem inclusa. Sem mensalidade de ferramenta, sem dependência de plataforma.

O que está incluso

Tudo o que entregamos

O processo começa com um briefing enviado ao cliente — perguntas sobre a empresa, a oferta e o público. A partir daí, construímos a página inteira: sem templates prontos, sem ajuste de tema.

Entregável Descrição Incluso
Design personalizado
 Layout desenvolvido com base na identidade visual do cliente — cores, tipografia, tom. Não usamos templates: cada página é construída do zero. ✓
Copy
 Textos criados a partir do briefing respondido pelo cliente. Headline, proposta de valor, prova social, CTA — estrutura pensada para converter. ✓
Mobile responsivo
 A página funciona perfeitamente em qualquer tamanho de tela — celular, tablet e desktop — sem perda de hierarquia ou legibilidade. ✓
Otimização de velocidade
 Código limpo, imagens otimizadas e carregamento rápido desde a entrega. Velocidade impacta diretamente o custo por resultado nas campanhas. ✓
Hospedagem
 Publicada na Vercel. Sem mensalidade para o cliente no volume padrão de acesso. ✓
Disparo de e-mail
 Integração com Resend inclusa quando necessário — confirmação de cadastro, entrega de material, notificações. Sem custo adicional no volume padrão. ✓
Infraestrutura — sem custo mensal no uso padrão
Vercel gratuito

Hospedagem da página. O plano gratuito cobre o volume padrão de tráfego — só migra para pago se o acesso escalar muito além do comum.

Resend gratuito

Disparo de e-mails transacionais. O plano gratuito cobre o volume padrão — só entra custo se o volume de envios for alto.

Sobre o briefing: antes de qualquer pixel, enviamos um formulário de briefing ao cliente. São as respostas que orientam o design, a copy e a estrutura da página — o resultado final reflete o negócio real, não uma página genérica.
